Link html element.

```html
&#60;xb-link
    id="..."
    class="..."    
    href="..."
    name="..."
    tabindex="..."
    target="_self|_blank|_parent|_top"
    theme="normal|outer|pseudo|input"
    disabled>&#60;/xb-link>
```
